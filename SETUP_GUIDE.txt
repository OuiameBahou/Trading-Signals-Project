================================================================================
  TRADING SIGNALS PROJECT — FULL SETUP GUIDE
  How to run this project on a new machine
================================================================================

--------------------------------------------------------------------------------
  SYSTEM REQUIREMENTS
--------------------------------------------------------------------------------

  Python   : 3.10.x  (https://www.python.org/downloads/release/python-3100/)
  Node.js  : 20.17.0 (https://nodejs.org/)  — NOTE: Vite 7 requires 20.19+
             Recommended: install 20.19+ or 22.x to avoid Vite warnings
  npm      : 10.8.2+ (bundled with Node.js)


--------------------------------------------------------------------------------
  PROJECT STRUCTURE
--------------------------------------------------------------------------------

  trading_signals_project/
  ├── lead_lag/
  │   ├── backend/          → Flask API (port 5000)
  │   └── frontend/         → React/Vite app (port 5173 dev / served by Flask prod)
  ├── FX/
  │   └── backend/          → FastAPI server (port 8001)
  └── sentiment_analysis/
      └── backend/          → NLP/Sentiment FastAPI (port 8002)


--------------------------------------------------------------------------------

  Flask API — port 5000
--------------------------------------------------------------------------------

                scipy==1.15.3 statsmodels==0.14.5 hmmlearn==0.3.3 \
                openpyxl==3.1.5 yfinance==0.2.66

  Or install from file:
    pip install -r lead_lag/backend/requirements.txt

--------------------------------------------------------------------------------
  5. START EVERYTHING  (recommended)
--------------------------------------------------------------------------------

  Just run start.bat from the project root:
    start.bat

  This opens 4 terminals:
    → Lead-Lag Backend   : python app.py           (port 5000)
    → FX Backend         : uvicorn main:app         (port 8001)
    → Sentiment Backend  : python main.py web       (port 8002)
    → Frontend Dev       : npm run dev              (port 5173)

  Then open:  http://localhost:5173


--------------------------------------------------------------------------------
  6. VIRTUAL ENVIRONMENT (recommended)
--------------------------------------------------------------------------------

  Create and activate a venv before installing:

    Windows:
      python -m venv venv
      venv\Scripts\activate
      pip install <packages>

    Mac/Linux:
      python3 -m venv venv
      source venv/bin/activate
      pip install <packages>

  TIP: Use one venv for the lead_lag + FX backends (they share the same
  packages), and a separate venv for sentiment_analysis (heavy ML deps).


--------------------------------------------------------------------------------
  7. COMMON ISSUES
--------------------------------------------------------------------------------

  "No module named flask"
    → Make sure you activated the right venv before running python app.py

  "uvicorn not found"
    → pip install uvicorn

  Vite warning about Node.js version
    → Upgrade Node.js to 20.19+ or 22.x

  Frontend showing old data after backend changes
    → Rebuild: cd lead_lag/frontend && npm run build
    → Hard refresh browser: Ctrl+Shift+R

  Port already in use
    → Kill the process using that port and restart
